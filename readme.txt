== Description ==

Displays a widget that let visitors vote decisions created on Becide.com, with an option to focus on those created by a given user (i.e. yourself).

== Installation ==

1. Upload `becide.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in your WordPress dashboard
1. Go to Apperance > Widgets screen
1. Drag Becide widget to desired sidebar
1. Configure settings

== Frequently Asked Questions ==

= How I can create votable decisions?

Go to [becide.com](http://www.becide.com/ "Decisions made easy"), create your decision and register.
Remember that decisions will be available for community vote once validated by Becide (in a few hours max).

= How I can show in the widget only my decisions?

Simply add your Becide username in the configuration option of the widget in the WordPress Widgets page.
